Implementation Notes:
1. Apply the 'TooltipHover' script to an object you want to detect Hover for the Tooltip
	a. Hover Text - Text to display in Tooltip
	b. Tooltip Box Prefab - Drag in the TooltipBox prefab. Customize prefab as you like
	c. New Parent - Which panel should the prefab spawn under. recommend a panel rendered on top
2. Enable Raycasting on the object this script is applied to
